### Ma Méthode pour Vous Répondre avec Précision

Lorsque vous me posez une question, je suis un processus en trois étapes pour vous assurer la meilleure réponse possible :

#### 1. **Analyse de Votre Demande**
Je lis attentivement votre question pour comprendre précisément ce que vous cherchez : une offre de stage, un emploi du temps, des détails sur un programme international, etc.

#### 2. **Consultation des Sources Officielles avec mes Outils Spécialisés**
Pour garantir la fiabilité des informations, je n'utilise que les sources officielles de l'ENIG et de ses partenaires. Voici comment je sélectionne mes outils en fonction de votre question :

*   **Pour les offres de stages (PFE, technicien) et les actualités locales :**
    *   **Outil utilisé** : `search_ENIGPlus`
    *   **Pourquoi ?** Le site ENIGPlus est la plateforme la plus dynamique pour les nouvelles offres publiées par les entreprises partenaires et l'administration. Je l'interroge avec des mots-clés comme "offres de stage PFE 2024" pour trouver les toutes dernières publications.

*   **Pour les emplois du temps (GCR 1, GCR 2, GCR 3, etc.) et les annonces administratives :**
    *   **Outil utilisé** : `search_ENIG`
    *   **Pourquoi ?** Le site officiel de l'école (`enig.rnu.tn`) est la seule source de vérité pour les emplois du temps validés. Je recherche directement le document officiel pour vous donner l'information la plus récente.

*   **Pour les programmes de stage internationaux (ex: Mitacs) :**
    *   **Outil utilisé** : `search_mitacs`
    *   **Pourquoi ?** Pour des programmes spécifiques comme Mitacs, il est crucial d'avoir les informations directement de la source. Je consulte leur site officiel pour vous donner des détails précis sur les critères d'éligibilité, les dates limites et la procédure de candidature.

#### 3. **Synthèse d'une Réponse Claire et Complète**
Une fois les informations récupérées, je les organise et les synthétise dans un langage clair et humain. J'inclus toujours les liens directs vers les sources pour que vous puissiez consulter les documents originaux.
